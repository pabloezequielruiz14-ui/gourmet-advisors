# Gourmet Advisors - Sitio Web Gastronómico

Sitio web profesional para asesoramientos culinarios y servicios gourmet.  
Incluye blog, galería, landing eventos y portafolio de recetas.

## Estructura del proyecto

```
assets/            # Imágenes, favicon y archivos estáticos
index.html         # Página principal
style.css          # Estilos personalizados
scripts.js         # Funciones JavaScript
blog1.html         # Blog: Artículo individual (ejemplo)
recetas.html       # Portafolio de recetas
evento.html        # Landing para evento especial
README.md          # Guía de uso
```

## Instalación rápida

1. Descarga los archivos y colócalos en tu servidor/local.
2. Sube imágenes personalizadas en `assets/`.
3. Abre `index.html` en el navegador para ver el sitio.
4. Opcional: Personaliza las rutas de redes y los textos.

## Personalización

- **Blog:** Agrega artículos nuevos (copiando blog1.html, cambiando contenido y nombre).
- **Galería:** Cambia las imágenes en `assets/` y edita los `<img>` en la galería.
- **Portafolio de recetas:** Agrega cards nuevas en `recetas.html`.
- **Landing de eventos:** Modifica el contenido de `evento.html`.
